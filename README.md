# Gestor de Bombas Renov

Plataforma de monitoramento e automação de poços, bombas e reservatórios da **Renov Tecnologia Agrícola**.

## Tecnologias

- Vite + React + TypeScript
- Tailwind CSS + shadcn/ui
- Electron (versão desktop .exe)

## Desenvolvimento local

```sh
npm install
npm run dev
```

Requer Node.js + npm instalados.
